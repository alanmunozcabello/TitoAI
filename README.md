# TitoAI: Local AI Assistant Learning Project

Welcome to the **TitoAI** project! This repository is a personal learning project dedicated to understanding and implementing the **Model Context Protocol (MCP)** within a local AI assistant. The goal is to build a powerful, privacy-focused, and highly capable assistant completely locally to learn AI integration hands-on in a fun, practical way.

## Project Purpose

The primary purpose of this project is to learn how to connect and orchestrate local AI models with external tools and real-world capabilities. By building this assistant, I aim to master tool-calling, API integrations, and local model deployment.

## Tech Stack

This project leverages cutting-edge open-source tools and frameworks:

*   **Local LLM Host:** [LM Studio](https://lmstudio.ai/) for serving models locally.
*   **LLM:** **Llama 3.1 8B Q4** - A highly capable model intentionally chosen for its excellent tool-calling and instruction-following abilities.
*   **Backend:** **FastAPI** - A modern, fast (high-performance) web framework for building APIs with Python.
*   **Core Technology:** **Model Context Protocol (MCP)** - To standardize how the AI assistant interacts with data sources and tools.
*   **Deployment & Orchestration:** **Docker**, **Minikube**, **Kubernetes**, and **k9s** - For containerizing the application and managing local cluster deployments.

## Planned Capabilities (Roadmap)

The vision for this assistant is to start simple and scale into a comprehensive personal aide. Planned features include:

### Document & File Processing
*   Read and extract information from various document formats (PDF, Excel, Word, PowerPoint, etc.).

### Computer Vision & Audio
*   Image description and analysis (built with future scalability in mind).
*   Camera interaction and real-time visual processing.
*   Voice interaction (speech-to-text and text-to-speech) to talk seamlessly with the assistant.

### Communication Interfaces
*   Seamless backend-to-frontend text messaging interface.

### Real-World Integrations
*   **WhatsApp Integration:** Ability to send messages and interact via WhatsApp.
*   **Google Workspace APIs:**
    *   Send and read emails via Gmail.
    *   Manage schedules and set events in Google Calendar.
    *   Take and organize notes automatically.

## Why This Project?

Building a local AI assistant from the ground up offers a unique opportunity to:
*   Gain hands-on experience with the **Model Context Protocol**.
*   Understand the intricacies of prompting and orchestrating tool usage with open-weight models like **Llama 3.1**.
*   Learn how to build scalable and robust backend architectures with **FastAPI**.
*   Learn containerization and local infrastructure deployments using the **Docker + Minikube + Kubernetes + k9s** stack.
*   Make the learning phase faster, highly practical, and fun by building a tool I can actually use.
